Team's Purpose

Our purpose is to build an interactive web application which will be used to play the treasure hunt challenge. The treasure hunt challenge is an online quiz game in which players answer questions by looking for clues in a real-life environment using an existing API that enables them to communicate with a server which stores the questions and answers.
